DISAPPOINTMENT

/ˈtrɪvɪəl/ adjective - of little value or importance.

/dɪsəˈpɔɪntm(ə)nt/ noun - sadness or displeasure caused by the non-fulfilment of one's hopes or expectations.

Trivial Disappointment is a last person standing trivia game that is free to play and can be run on any internet ready smart device. The rules are simple, be the last one remaining or the fastest in the game. Answer just one wrong question and you are out the game (In which case you should feel bad). The game ends when there is one last person in the game or in the case of a tie the fastest. This person will be the ultimate champion of knowledge whilst the loser(s) will be trivially disappointed. Questions are split into categories and are complety random thanks to the free to use trivia database API

 

 

 

Easy to play - Players can use an internet ready smart device and join / host a room.

Get a question wrong and you lose the game.


Choose a category - Hosts can choose out of 23 categories to tailor games.

Or choose to play questions from all categories.


Games modes can be selected - Fancy a quick game?

Hosts can select games with 1 question or 50 where the fastest player wins

Trivial Disappointment - Dissapointing people since 2020

Players will be competing in a trivia last person standing game - the game will be hosted on a smart device and players will use a room code to join a game. If they lose they will be removed from that room until another game begins. The game ends when one player remains in the room.

  • Players - in the game will be given a random username which will be taken from a database of random colours and animals. The last user still in the game wins. This way no users have no custom input when playing with others.

  • Currently 3,558 Verified Questions - and 3,508 questions currently pending with the Open Trivia Database API. Meaning the game will continually grow alonside the API as users add more verified questions. https://opentdb.com/api_config.php

  • Requirements -currently the game is free and can be played through wifi. A host will start a room and players will join that room with the hosts code.

  • The Reason for making this -each team member making TRIVIAL DISAPPOINTMENT gets together weekly to play couch co-op games. We miss the age of playing games in the same room so we decided to go ahead with this project which can be played locally or online
